CSS

div

É um elemento de bloco, o que significa que ele ocupa toda a largura disponível e vem com linha pausas antes e depois.

marge

As propriedades de margem CSS são usadas para criar espaço em torno de elementos, fora de quaisquer fronteiras definidas.

padding

São propriedades de preenchimento e CSS são usadas para gerar espaço ao redor conteúdo de um elemento, dentro de qualquer borda definida.
